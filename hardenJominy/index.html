📐 计算原理说明（基于ASTM A255标准）
1. 初始硬度 IH（100%马氏体硬度）
  IH = 33.087 + 50.723C + 33.662C² - 2.7C³ - 107C⁴ + 43.5C⁵ (HRC)
  • 仅与碳含量相关，是钢能达到的最高硬度

2. 理想临界直径 DI（淬透性指标）
  DI = 25.4 × f(C) × √(fMn × fSi × fCr × fNi × fMo) × fB (mm)
  • DI越大 → 淬透性越好 → 更大截面可完全淬硬
  • 各元素乘法因子: fMn=1+3.34Mn, fCr=1+2.16Cr, fMo=1+3Mo

3. Ms/Mf温度（Andrews公式）
  Ms = 539 - 423C - 30.4Mn - 17.7Ni - 12.1Cr - 7.5Mo (°C)
  Mf = Ms - 215 (°C)
  • Ms: 马氏体转变开始温度 | Mf: 马氏体转变结束温度

📐 Jominy末端淬火试验原理（ASTM A255 / ISO 642）
1. 试验原理
  标准试样(Φ25.4×102mm)加热奥氏体化后，一端喷水淬火
  距淬火端越远 → 冷却速度越慢 → 硬度越低

2. 硬度预测模型
  HRC(J) = HRCbase + (IH - HRCbase) × e-k·J
  • J = 距离淬火端的距离(mm)
  • k = 衰减系数 = 0.06 / √(DI/10)，DI越大，k越小
  • HRCbase = 0.2×IH（完全退火态硬度）

3. 物理意义
  • DI大的钢 → 临界冷却速度低 → 曲线平缓 → 淬透性好
  • DI小的钢 → 临界冷却速度高 → 曲线陡峭 → 淬透性差
  ✓ 相同碳含量(IH相同)，不同合金含量 → 不同曲线形状

📐 淬硬深度预测原理（Lamont-Grossman方法）
1. Grossman临界直径转换
  Dc = DI × (1 - e-1.5H) (近似公式)
  • H=∞时Dc→DI | H=1(水淬)时Dc≈0.78DI | H=0.5(油淬)时Dc≈0.53DI

2. Lamont等效Jominy距离
  圆棒各位置(r/R)对应一个等效Jominy距离Jeq
  Jeq = Jcenter × (1 - r/R) (简化模型)
  • r/R=0(中心): Jeq=Jcenter | r/R=1(表面): Jeq≈0

3. 各位置硬度计算
  HRC(r/R) = Jominy曲线在Jeq处的硬度
  • 直接复用Jominy硬度预测模型
  ✓ DI不同 → Jominy曲线不同 → 硬度分布不同
