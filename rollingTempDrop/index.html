计算公式
辐射温降（Stefan-Boltzmann）：
ΔT_rad = σ·ε_r·(T₀⁴-T_env⁴)·t / (ρ·c·h/2)

变形升温：
ΔT_def = η·K_f·10⁶·ε / (ρ·c)

最终温度：
T_final = T₀ - ΔT_rad + ΔT_def

不含轧辊接触导热、喷淋冷却和对流换热
参数说明
σ = 5.67×10⁻⁸ W/(m²·K⁴) — Stefan-Boltzmann 常数

ε_r ≈ 0.8 — 钢的辐射率

ρ ≈ 7800 kg/m³ — 钢的密度

c ≈ 600 J/(kg·°C) — 比热容

η ≈ 0.9 — 机械功转热效率
应用场景
热轧温度控制 — 预测道次间温度变化

终轧温度预测 — 确保终轧温度满足工艺要求

道次间温降粗估 — 优化轧制节奏与冷却策略
