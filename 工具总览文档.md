## 1. 锻比计算 (SegmentRatio)

### 用途
计算锻造过程中的**锻造比 K**，用于评估锻造变形量是否满足锻件质量要求。锻比直接影响锻件内部缺陷（铸态组织、疏松、偏析等）的改善程度。

### 原理与公式

支持三种锻造方式：

**拔长锻造（最常用）：**
```
K = A₀ / A₁ = (D₀ / D₁)²
```
- A₀：锻前截面积，A₁：锻后截面积
- D₀：锻前直径，D₁：锻后直径
- 对于圆截面：A = π/4 × D²，因此 A₀/A₁ = (D₀/D₁)²

**镦粗锻造：**
```
K = H₀ / H₁
```
- H₀：锻前高度，H₁：锻后高度

**多道次锻造：**
```
K = K₁ × K₂ × K₃
```
- 各道次锻比的乘积即为总锻比

### 应用场景
- 大型锻件（轧辊、主轴、转子等）工艺设计
- 判断锻比是否满足标准要求（通常要求 K ≥ 3~4）
- 多火次锻造方案的总变形量评估

### ⚠️ 代码问题
**拔长锻造的计算逻辑有设计缺陷**：代码同时要求输入 A₀/A₁ 和 D₀/D₁，然后取两个结果的平均值 `(A₀/A₁ + (D₀/D₁)²) / 2`。但对于圆截面，这两个表达式理论上相等——取平均没有意义；对于非圆截面，(D₀/D₁)² 本身就不适用。建议：应让用户选择"按截面积"或"按直径"其中一种方式输入。

## 2. 碳当量(CE)计算器 (carbonEquivalent)

### 用途
根据钢材化学成分计算**碳当量**，评估钢材的**焊接性能**。碳当量越高，焊接冷裂纹倾向越大，需要预热/后热等控制措施。

### 原理与公式

工具同时计算三种碳当量指标：

**CE(IIW) — 国际焊接学会公式（适用于中高碳钢）：**
```
CE(IIW) = C + Mn/6 + (Cr + Mo + V)/5 + (Ni + Cu)/15
```

**CEQ(JIS) — 日本工业标准公式：**
```
CEQ = C + Mn/6 + Si/24 + Ni/40 + Cr/5 + Mo/4 + V/14
```

**Pcm — 冷裂敏感系数（Ito-Bessyo公式，适用于低碳微合金钢）：**
```
Pcm = C + Si/30 + (Mn + Cu + Cr)/20 + Ni/60 + Mo/15 + V/10 + 5B
```
注：B（硼）输入单位为 ppm，代码内部已转换为百分比（÷10000）。

**焊接性评价标准（基于 CE(IIW)）：**

| CE 范围 | 评价 |
|---------|------|
| ≤ 0.40 | 焊接性能优秀，一般不需预热 |
| 0.40 ~ 0.60 | 焊接性能良好，需采取控制措施 |
| 0.60 ~ 0.80 | 焊接困难，需严格焊接工艺 |
| > 0.80 | 极难焊接，通常不建议焊接 |

### 应用场景
- 焊接工艺评定（WPS）中的预热温度确定
- 钢材选材时的焊接性评估
- 管线钢、桥梁钢、压力容器钢的质量控制

### ⚠️ 代码问题
**HTML页面显示的公式与代码实际计算不匹配**：
1. 页面显示的"CEV"公式实际写的是 Pcm 公式形式，且出现 `DOUDLE` 拼写错误（应为 DOUBLE 或根本不需要此前缀）
2. 页面显示的"PCM"公式写成了 `C + Mn/6 + Ni/15 + Cr/5 + Cu/15 + Mo/5 + V/5`，这与代码实际计算的 Pcm 完全不同
3. **代码中三个公式的实际计算是正确的**，仅页面展示文字有误，会误导使用者

## 3. 淬透性与Jominy曲线计算器 (hardenJominy)

### 用途
基于钢材化学成分，计算**理想临界直径 DI**、预测**末端淬火（Jominy）硬度曲线**，并评估圆棒工件的**淬硬深度分布**。这是热处理工艺设计中最核心的工具之一。

### 原理与公式

工具包含三个子模块：

#### 模块一：淬透性基础参数

**初始硬度 IH（100%马氏体硬度，ASTM A255多项式）：**
```
IH = 33.087 + 50.723C + 33.662C² − 2.7048C³ − 107.02C⁴ + 43.523C⁵  (HRC)
```

**理想临界直径 DI（ASTM A255）：**
```
DI = 25.4 × f(C) × √(f_Mn × f_Si × f_Cr × f_Ni × f_Mo) × f_B  (mm)
```
各元素乘法因子：
- f_Mn = 1 + 3.34Mn
- f_Si = 1 + 0.7Si
- f_Cr = 1 + 2.16Cr
- f_Ni = 1 + 0.363Ni
- f_Mo = 1 + 3.0Mo
- f(C) 为碳含量分段函数（5段）

碳因子 f(C) 分段定义：
| C 范围 | f(C) |
|--------|------|
| C ≤ 0.39 | 0.54C |
| 0.39 < C ≤ 0.55 | 0.171 + 0.001C + 0.265C² |
| 0.55 < C ≤ 0.65 | 0.115 + 0.268C − 0.038C² |
| 0.65 < C ≤ 0.75 | 0.143 + 0.2C |
| C > 0.75 | 0.062 + 0.409C − 0.135C² |

硼因子 f_B（仅在 0 < B ≤ 0.005% 时有效）：
```
f_B = 1 + (1 − 1.5C) × min(1, 0.5 + 0.5/√alloyFactor) × (B/0.002)
f_B 上限为 2.5
```

**马氏体转变温度（Andrews公式）：**
```
Ms = 539 − 423C − 30.4Mn − 17.7Ni − 12.1Cr − 7.5Mo  (°C)
Mf = Ms − 215  (°C)
```

#### 模块二：Jominy末端淬火曲线

使用指数衰减模型：
```
HRC(J) = HRC_base + (IH − HRC_base) × e^(−k·J)
```
- J：距淬火端的距离（mm）
- k = 0.06 / √(DI/10)，衰减系数
- HRC_base = 0.2 × IH（完全退火态硬度）
- DI 越大，k 越小，硬度衰减越慢（淬透性越好）

#### 模块三：淬硬深度预测（Lamont-Grossman方法）

```
Dc = DI × (1 − e^(−1.5H))  (临界直径)
```
- H：淬火介质烈度（水=1.0, 盐水=2.0, 快速油=0.7, 中速油=0.5, 空气=0.02）

圆棒各径向位置的硬度通过等效Jominy距离查曲线获得。

### 应用场景
- 钢种选择：根据零件尺寸反推所需DI，选择合适钢种
- 热处理工艺设计：预测淬火后硬度分布
- 质量控制：对比实测Jominy曲线与预测曲线

### 公式验证
代码中的公式与 ASTM A255 / Andrews 标准公式一致，**无重大公式错误**。

## 4. 硬度转化工具 (hardness)

### 用途
在四种常用硬度标尺（努氏HK、维氏HV、布氏HB、洛氏HRC）之间进行互相转换。

### 原理与公式

基于 **ISO 18265** 标准的经验回归公式，部分典型转换如下：

| 转换方向 | 公式 | 适用范围 |
|----------|------|----------|
| HV → HRC | 116 − 1500/√HV | HRC 20~65 |
| HB → HRC | 116 − 1460/√HB | HRC 20~65 |
| HRC → HV | 0.16x² + 13.5x + 113 | HRC 30~60 |
| HRC → HB | 0.15x² + 12.8x + 107 | HRC 30~60 |
| HK → HV | HK × 0.962 | 全范围 |
| HV → HB | HV × 0.95 | HV < 600 |
| HK → HRC | 115 − 215000/HK^1.2 | HK > 250 |

### 应用场景
- 不同检测设备间的硬度值互换
- 质检报告中硬度标尺的统一
- 材料力学性能的快速估算（如 HB ≈ 3.45 × σ_b）

### 公式验证
公式为经验回归模型，在各自标注的适用范围内与ISO 18265查表数据吻合良好。**无重大公式问题。**

## 5. 密度计算 (midu)

### 用途
根据合金元素含量和温度，计算钢铁材料的**密度**。

### 原理与公式

**室温合金密度修正：**
```
ρ₂₀ = 7.85 + Σ(ki × Ci)  (g/cm³)
```
- 7.85 为纯铁基础密度
- ki 为各元素的密度影响系数，Ci 为元素含量（wt%）

代码中的 ki 值：

| 元素 | ki (g/cm³ per 1%) | 效果 |
|------|-------------------|------|
| Cr | +0.03 | 增大 |
| Ni | +0.012 | 增大 |
| Mo | +0.015 | 增大 |
| Mn | +0.005 | 增大 |
| W  | +0.1 | 显著增大 |
| Al | −0.05 | 减小 |
| Si | −0.008 | 减小 |
| C  | −0.06 | 减小 |

**高温密度修正（热膨胀）：**
```
ρ(T) = ρ₂₀ / [1 + 3α(T − 20)]
```
- α = 1.2×10⁻⁵ /°C（线膨胀系数）
- 3α 是体膨胀近似

### 应用场景
- 钢材理论重量计算（配合"钢材重量计算"工具使用）
- 铸造/锻造工艺的充型计算
- 高温下的热力学/传热计算

### ⚠️ 代码问题（需确认）
**Cr 的 ki 系数符号存疑**：纯 Cr 密度 7.19 g/cm³ 低于 Fe 的 7.87 g/cm³，理论上添加 Cr 应降低合金密度（ki 为负值）。多数参考文献给出 Cr 的 ki ≈ −0.03。代码中使用 **+0.03** 可能使高铬钢（如不锈钢）的密度计算结果偏高。建议对照厂内使用的参考手册确认此系数。

## 6. 淬火温度计算器 (quenchTemCal)

### 用途
根据碳含量和钢种类别，计算**临界温度（Ac1、Ac3）**和推荐淬火温度范围；高级模式可计算**马氏体转变温度（Ms/Mf点）**。

### 原理与公式

**基础计算模块：**
```
Ac1 = 727°C（固定值，钢铁相变基本常数）
Ac3 = 912 − 203√C  (°C)（亚共析钢经验公式）
```

淬火温度推荐逻辑：
- 亚共析钢（C < 0.77%）：Ac3 + 30~50°C
- 共析钢（C = 0.77%）：Ac1 + 30~50°C
- 过共析钢（C > 0.77%）：Ac1 + 30~50°C

合金钢温度上调：低合金+20°C，中合金+30°C，高合金+40°C

**高级模块 — Ms/Mf 计算（Andrews公式）：**
```
Ms = 539 − 423C − 30.4Mn − 17.7Ni − 12.1Cr − 7.5Mo  (°C)
Mf = Ms − 200  (°C)
```

### 应用场景
- 新钢种热处理工艺开发
- 淬火温度参数的快速查询
- Ms 点评估：判断是否需要分级淬火或等温淬火

### ⚠️ 代码问题
代码计算了推荐淬火温度范围 `tempRange`（如"787~807°C"），但**未在输出区域显示**——用户只能看到 Ac3 温度，看不到推荐的淬火温度范围，降低了工具的实用价值。

## 7. 单位制度转化 (unit)

### 用途
冶金行业常用的**七大物理量**（温度、长度、面积、体积、力、压力、质量）的单位互转。

### 原理与公式

覆盖 7 类共 28 个单位之间的双向转换，总计约 80 个转换规则。核心转换举例：

| 类别 | 转换示例 | 公式 |
|------|----------|------|
| 温度 | °C → °F | F = C × 9/5 + 32 |
| 温度 | °C → K | K = C + 273.15 |
| 长度 | mm → in | in = mm / 25.4 |
| 压力 | MPa → psi | psi = MPa × 145.038 |
| 压力 | MPa → ksi | ksi = MPa × 0.145038 |
| 力 | kN → tf | tf = kN / 9.81 |
| 质量 | kg → lb | lb = kg × 2.20462 |

### 应用场景
- 国际贸易中公制/英制单位转换
- 对照国外标准时的单位换算
- 日常冶金工程计算辅助

### ⚠️ 代码BUG（严重）
**面积转换 mm² → m² 公式错误**：
```javascript
'mm2→m2': value => value / 10000   // ← 错误！
'm2→mm2': value => value * 10000   // ← 错误！
```
正确关系：**1 m² = 1,000,000 mm²**（10⁶），当前代码除以 10,000（10⁴），**结果会偏大 100 倍**。应修改为：
```javascript
'mm2→m2': value => value / 1000000
'm2→mm2': value => value * 1000000
```

## 8. 钢材重量计算 (weight)

### 用途
根据钢材的**截面形状参数、长度和密度**，计算各类型材/板材的理论重量。

### 原理与公式

支持 15 种截面类型，核心公式为 **W = V × ρ / 1000**（体积×密度，mm³→g）

各截面体积公式：

| 钢材类型 | 体积公式 V (mm³) |
|----------|------------------|
| 圆钢 | π(d/2)² × L |
| 方钢 | d² × L |
| 钢板/扁钢 | b × d × L |
| 六角钢 | 0.866 × s² × L |
| 八角钢 | 0.828 × s² × L （**代码实际写为 8.28**） |
| 螺纹钢 | πd²/4 × L |
| 等边角钢 | [d(b−d)×2 + d² + (1−π/4)R²] × L |
| 槽钢 | [hb − (h−2t)(b−d) + 2(1−π/4)(R²−r²)] × L |
| 工字钢 | [hb − (h−2t)(b−2d) + (R²−r²)(4−π)] × L |
| H型钢 | [hb − (h−2t)(b−2d) + R²(4−π)] × L |
| 钢管 | π/4 × (D² − (D−2s)²) × L |
| 不等边角钢 | [d(b+B−d) + R²(1−π/4)] × L |
| 方管 | [w₁w₂ − (w₁−2c)(w₂−2c)] × L |
| 花纹板 | b × d × L × 1.0703 |

### 应用场景
- 钢材采购的理论重量核算
- 结构设计中的自重计算
- 材料成本估算

### ⚠️ 代码BUG（严重）
**八角钢体积系数错误**：
```javascript
volumeFormula: (s, L) => Math.pow(s, 2) * L * 8.28  // ← 错误！
```
正八边形（s 为对边距）截面积 = 2(1+√2) × [s·tan(π/8)]² ≈ **0.8284 × s²**。

当前系数 **8.28** 是正确值 **0.828** 的 10 倍（小数点位置错误），**计算出的八角钢重量会偏大 10 倍**。

## 9. 供氧强度/吹炼节奏计算器

### 用途
计算转炉吹炼的**平均氧气流量 Q** 和**供氧强度 I**，用于评估"吹得猛不猛"。支持按总耗氧量或分段吹炼计划两种模式。

### 原理与公式

```
Q = V / t          (Nm³/min，平均氧气流量)
I = Q / T          (Nm³/min/t，吨钢供氧强度)
```
- V：总耗氧量（Nm³）
- t：吹炼时间（min）
- T：炉次吨位（t）

分段模式：
```
V = Σ(Qi × ti)    (各段流量×各段时间的累加)
t = Σti
```

### 应用场景
- 转炉吹炼工艺参数设计与对比
- 不同吨位/炉型间的供氧强度横向比较
- 吹炼节奏（分段流量）的方案优化

### 公式验证
纯量纲计算，**无公式问题**。代码在 I < 3 或 I > 12 时给出经验区间告警，合理。

## 10. 冷却剂/矿石温降计算器

### 用途
基于能量守恒，估算"**加多少冷却剂能降多少温**"或"**要降到目标温度需要加多少冷却剂**"。

### 原理与公式

**钢水需要释放的热量：**
```
Q_need = m_steel(kg) × Cp_steel(kJ/kg·K) × (T_current − T_target)
```

**冷却剂每 kg 吸收的热量 q：**

仅加热模式（矿石/氧化物）：
```
q = Cp_s × (T_final − T₀) + H_reaction
```

加热+熔化+过热模式（废钢/金属料）：
```
q = Cp_s × (Tm − T₀) + L + Cp_l × (T_final − Tm) + H_reaction
```
- Cp_s：固相比热，Cp_l：液相比热
- Tm：熔点，L：熔化潜热
- H_reaction：反应吸/放热（正=吸热，负=放热）

```
m_coolant = Q_need / q
```

## 11. 合金加入量计算器

### 用途
根据"当前成分→目标成分"的差值，计算需要补充的**合金加入量**，并给出成本估算和回收率不确定度下的加入量区间。

```
m_element_needed (kg) = (target% − current%) / 100 × m_steel(kg)
```

```
m_low  = m_element / (content × (recovery + unc))
```

## 12. 渣氧化性口径换算与判级

### 用途
在转炉/精炼渣分析中，将不同表达口径（FeO%、Fe₂O₃%、ΣFeO%、TFe%）互相换算，并根据阈值进行氧化性等级判定。

### 原理与公式

**ΣFeO（FeO等效，all-iron method）：**
```
ΣFeO = FeO + 0.9 × Fe₂O₃
```
其中 0.9 = M(2FeO) / M(Fe₂O₃) = 144/160

**TFe（以Fe元素计）：**
```
TFe = FeO × (56/72) + Fe₂O₃ × (112/160)
```
- 56/72 = M(Fe)/M(FeO) = 0.778
- 112/160 = M(2Fe)/M(Fe₂O₃) = 0.700

**近似反推：**
```
ΣFeO ≈ TFe × (72/56)    （假设全部按FeO口径）
TFe ≈ ΣFeO × (56/72)     （反向）
```

**判级（用户自定义阈值）：**
- 低：判级值 < 阈值1
- 中：阈值1 ≤ 判级值 ≤ 阈值2
- 高：判级值 > 阈值2

### 应用场景
- 转炉终渣氧化性控制（影响脱磷效果、钢水氧含量）
- 不同化验口径间的统一比较
- 渣系优化方案评估

### 公式验证
化学计量换算正确，0.9 系数推导无误。**无公式问题。**

## 13. 连铸壳厚/拉速安全核算

### 用途
使用经典的"**平方根定律**"计算连铸结晶器出口的凝固壳厚度，核算当前拉速是否安全（壳厚是否满足最低要求），并反推允许的最大拉速。

### 原理与公式

**停留时间：**
```
t = L / v  (min)
```
- L：结晶器有效长度（m）
- v：拉速（m/min）

**壳厚（平方根定律）：**
```
δ = K × √t = K × √(L/v)  (mm)
```
- K：凝固系数（mm/√min），与钢种、过热度、冷却强度等有关

**反推最大拉速：**
```
v_max = L × (K / δ_min)²  (m/min)
```

**反推所需结晶器长度：**
```
L_req = v × (δ_min / K)²  (m)
```

### 应用场景
- 连铸拉速上限的安全核算（防漏钢）
- 不同钢种/断面切换时的拉速调整方案
- 结晶器设计（有效长度的确定）

### 公式验证
经典凝固传热经验公式，推导正确。**无公式问题。**

## 14. 造渣（石灰/白云石）计算器

### 用途
在给定 SiO₂ 负荷和渣量的前提下，计算**石灰和白云石的加入量**，使渣系碱度 B 和 MgO% 同时落入目标窗口。

### 原理与公式

**Si → SiO₂ 推算（可选）：**
```
Si_removed (kg) = (Si_in% − Si_target%) / 100 × m_steel(kg)
SiO₂ (kg) = Si_removed × (60/28)
```
M(SiO₂)/M(Si) = 60/28 ≈ 2.143

**碱度定义与目标 CaO：**
```
B = CaO / SiO₂     （质量比碱度）
CaO_required = B_target × SiO₂  (kg)
```

**目标 MgO：**
```
MgO_required (kg) = MgO_target% / 100 × m_slag (kg)
```

```
CaO_required = L × CaO_eff + D × CaO_dolo

```
解出：L（石灰 kg）、D（白云石 kg）

### 应用场景
- 转炉造渣方案设计（碱度与MgO的平衡）
- 石灰/白云石用量的快速估算
- 不同原料（CaO/MgO含量变化）的加料方案对比

### 公式验证
化学计量与线性代数联立方程，**公式正确**。

## 15. 轧制力计算器 (rollingForce)

### 用途
计算热轧过程中的**轧制力 P**。最新版代码已将公式统一收口到 `shared/rolling-formulas.js`，页面只负责输入、校验和展示结果。

### 原理与公式

**压下量与压下率：**
```
Δh = H − h
ε = Δh / H
```
- H：入口厚度（mm）
- h：出口厚度（mm）

**接触弧长：**
```
l_d = √(R × Δh)  (mm)
```

**应力状态系数（Ekelund 形式）：**
```
Q_p = 1 + 1.6μl_d/(H+h) − 1.2Δh/(H+h)
```

**轧制力：**
```
P = K_f × Q_p × b × l_d / 1000  (kN)
```
- K_f：变形抗力（MPa）
- b：轧件宽度（mm）
- 当 `Q_p ≤ 0` 时，代码直接拒绝计算，避免输出负轧制力这种鬼结果。

### 默认值（热轧中板典型工况）

| R | 450 mm | 工作辊半径 |
| H | 30 mm | 入口厚度 |
| h | 22 mm | 出口厚度 |
| b | 1200 mm | 板宽 |
| K_f | 150 MPa | 低碳钢热轧典型值 |
| μ | 0.35 | 热轧典型摩擦系数 |

### 应用场景
- 热轧板带轧制力估算
- 型钢、棒线材各道次轧制力粗算
- 轧机能力、电机功率、轧辊强度初步校核

### 公式验证
当前代码与页面公式一致，并补充了物理域校验。**公式链可用于工程估算，但仍不是生产级在线轧制力模型。**

## 16. 压下分配估算器 (reductionSchedule)

### 用途
给定坯料厚度、成品厚度和道次数，做多道次厚度的**几何分配估算**。注意：它现在明确叫“压下分配估算器”，不再假装是完整“压下规程”。真正规程还必须校核咬入、轧制力、力矩、功率、温度和设备极限。

### 原理与公式

**等压下率分配：**
```
r = h_f / H₀
ε_avg = 1 − r^(1/n)
h_i = H_i × (1 − ε_avg)
```

**递减真应变分配：**
```
ε_total = ln(H₀ / h_f)
w_i = n − i + 1
ε_i = ε_total × w_i / Σw_i
h_i = H_i × exp(−ε_i)
```
- 这不是直接按各道次压下率递减，而是把**总真应变**按权重递减分到各道次。
- 页面文案已经同步改为“递减真应变”，避免前台说一套、后台算一套。

**各道次输出：**
```
Δh_i = H_i − h_i
ε_i(压下率) = Δh_i / H_i
λ_i = H_i / h_i
累计压下率 = (H₀ − h_i) / H₀
```

**汇总参数：**
```
λ_total = H₀ / h_f
总压下率 = (H₀ − h_f) / H₀
```

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| H₀ | 230 mm | 坯料厚度 |
| h_f | 12 mm | 成品厚度 |
| n | 7 | 道次数，必须为整数 |
| 分配方式 | 等压下率 | 可选“等压下率”或“递减真应变” |

### 应用场景
- 初步生成厚度分配表
- 对比等压下率与递减真应变分配
- 为后续轧制力、咬入、温度和设备能力校核提供输入

### 公式验证
等压下率和递减真应变分配都能严格回到目标厚度。代码已拒绝小数道次数，避免 `7.9` 被偷偷截断成 `7` 这种蠢事。

## 18. 变形抗力计算器 (deformResistance)

### 用途
根据碳含量、轧制温度、应变和应变速率，计算热轧过程中的**变形抗力 K_f**。这是轧制力计算最关键的输入之一。

### 原理与公式

最新版已经修掉旧实现中的 Misaka 指数错误和过度简化 Shida 公式。现在页面调用共享公式：

**Misaka 公式：**
```
T_K = T + 273.15
A = 0.126 − 1.75C + 0.594C² + (2851 + 2968C − 1120C²)/T_K
K_f = exp(A) × ε^0.21 × ε̇^0.13 × 9.80665  (MPa)
```
- C：碳含量（%）
- T：轧制温度（°C）
- ε：真应变
- ε̇：应变速率（s⁻¹）

**Shida 公式（当前代码实现）：**
```
t = (T + 273.15) / 1000
n = 0.41 − 0.07C
t_d = 0.95(C + 0.41)/(C + 0.32)
```

当 `t ≥ t_d`：
```
g = 1
t_x = t
m = (−0.019C + 0.126)t + (0.075C − 0.05)
```

当 `t < t_d`：
```
g = 30(C+0.9)[t − 0.95(C+0.49)/(C+0.42)]² + (C+0.06)/(C+0.09)
t_x = t_d
m = (0.081C − 0.154)t + (−0.019C + 0.207) + 0.027/(C+0.32)
```

最终：
```
σ_f = 0.28g × exp(5/t_x − 0.01/(C+0.05))
K_f = 2/√3 × σ_f × [1.3(ε/0.2)^n − 0.3(ε/0.2)] × (ε̇/10)^m × 9.80665
```

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| C | 0.15% | 低碳钢 |
| T | 1000°C | 热轧典型温度 |
| ε | 0.3 | 单道次典型真应变 |
| ε̇ | 10 s⁻¹ | 热轧典型应变速率 |

### 应用场景
- 为轧制力计算器提供 K_f 输入
- 比较 Misaka 与 Shida 两个经验模型
- 判断温度、应变、应变速率变化对变形抗力的影响

### 公式验证
旧版文档和旧代码把 Misaka 的应变/应变速率指数写错了，这会明显低估 K_f。当前代码已按共享公式修正，并加入温度范围约束（600~1400°C）。**最新版无已知公式错误，但仍属于经验模型。**

## 19. 咬入条件计算器 (biteCondition)

### 用途
判断轧辊能否顺利**咬入轧件**。当压下量过大或摩擦系数过小时，轧件会打滑，无法进入辊缝。

### 原理与公式

**咬入角：**
```
α = arccos(1 − Δh/(2R))
```
- R：工作辊半径（mm），不是直径。
- 代码要求 `0 < Δh < 2R`，避免出现超过 90° 的非物理咬入角。

**摩擦角：**
```
β = arctan(μ)
```

**咬入判据：**
```
α ≤ β  → 可以咬入
α > β  → 不能咬入
```

**最大允许压下量：**
```
Δh_max = 2R × (1 − cosβ)
```

**按温度估算热轧摩擦系数：**
```
μ = 0.8 × (1.05 − 0.0005T)
```
- 当前共享公式把该经验式限制在 600~1400°C 内使用。

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| R | 450 mm | 工作辊半径 |
| Δh | 8 mm | 压下量 |
| 摩擦系数来源 | 手动输入 | 可按温度估算 |
| μ | 0.35 | 热轧典型值 |
| T | 1000°C | 按温度估算时使用 |

### 应用场景
- 大压下咬入安全校核
- 新规格试轧前判断压下量是否过猛
- 润滑/温度变化对咬入能力的影响评估

### 公式验证
几何关系和摩擦角判据正确；最新版修正了“辊径/半径”表述，并把非物理压下量直接拒绝。**无已知公式错误。**

### 应用场景

## 21. 前滑值与出口速度计算器 (forwardSlip)

### 用途

### 原理与公式

**由实测出口速度反算前滑：**
```
S_h = v_out / v_R − 1
```

```
v_out = v_R × (1 + S_h)
```

**输入互斥规则：**
- 有实测出口速度时：用 `v_out / v_R − 1` 反算前滑。
- 没有实测出口速度时：用输入的目标前滑值计算出口速度。
- 两者不能同时填，否则页面拒绝计算，避免静默覆盖输入。

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| v_R | 3.5 m/s | 辊面线速度 |
| v_out | 空 | 可选，实测出口速度 |
| S_h | 5% | 无实测出口速度时使用 |

### 应用场景
- 连轧速度匹配
- 根据实测出口速度反算前滑
- 根据目标前滑估算出口速度

### 公式验证
当前使用的是前滑定义式和速度换算式，物理含义清楚。旧版几何压下近似式已从页面删除；那个近似缺少摩擦和中性角，不适合在这个工具里伪装成通用前滑公式。

## 22. 热膨胀/冷缩量计算器 (thermalExpansion)

### 用途
计算钢材从热态冷却到室温后的**尺寸收缩量**，用于从热态测量尺寸推算冷态成品尺寸。这是热轧成品尺寸精度控制的基础。

### 原理与公式

**冷态尺寸：**
```
L_cold = L_hot / (1 + α × (T_hot − T_cold))  (mm)
```
- L_hot：热态测量尺寸（mm）
- T_hot：热态温度（°C）
- T_cold：冷态温度（°C），通常取室温 20°C
- α = 12.5 × 10⁻⁶ /°C（钢的线膨胀系数）

**收缩量与收缩率：**
```
ΔL = L_hot − L_cold  (mm)
收缩率 = ΔL / L_hot × 100%
```

**工程经验：**
- 900°C 时收缩率约 1.1%
- 即热态 1000mm 的板材冷到室温约缩短 11mm

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| L_hot | 1000 mm | 热态尺寸 |
| T_hot | 900°C | 终轧/卷取温度 |
| T_cold | 20°C | 室温 |

### 应用场景
- 热轧成品宽度/厚度的冷态换算
- 精轧辊缝设定时预留收缩余量
- 在线测厚/测宽仪的热态→冷态修正

### 公式验证
线膨胀公式为经典热力学公式，α = 12.5×10⁻⁶ /°C 是碳钢在 20~900°C 范围内的平均值，工程上广泛采用。**无公式问题。**
