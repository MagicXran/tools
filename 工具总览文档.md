## 1. 锻比计算 (SegmentRatio)

### 用途
计算锻造过程中的**锻造比 K**，用于评估锻造变形量是否满足锻件质量要求。锻比直接影响锻件内部缺陷（铸态组织、疏松、偏析等）的改善程度。

### 原理与公式

支持三种锻造方式：

**拔长锻造（最常用）：**
```
K = A₀ / A₁ = (D₀ / D₁)²
```
- A₀：锻前截面积，A₁：锻后截面积
- D₀：锻前直径，D₁：锻后直径
- 对于圆截面：A = π/4 × D²，因此 A₀/A₁ = (D₀/D₁)²

**镦粗锻造：**
```
K = H₀ / H₁
```
- H₀：锻前高度，H₁：锻后高度

**多道次锻造：**
```
K = K₁ × K₂ × K₃
```
- 各道次锻比的乘积即为总锻比

### 应用场景
- 大型锻件（轧辊、主轴、转子等）工艺设计
- 判断锻比是否满足标准要求（通常要求 K ≥ 3~4）
- 多火次锻造方案的总变形量评估

### ⚠️ 代码问题
**拔长锻造的计算逻辑有设计缺陷**：代码同时要求输入 A₀/A₁ 和 D₀/D₁，然后取两个结果的平均值 `(A₀/A₁ + (D₀/D₁)²) / 2`。但对于圆截面，这两个表达式理论上相等——取平均没有意义；对于非圆截面，(D₀/D₁)² 本身就不适用。建议：应让用户选择"按截面积"或"按直径"其中一种方式输入。

## 2. 碳当量(CE)计算器 (carbonEquivalent)

### 用途
根据钢材化学成分计算**碳当量**，评估钢材的**焊接性能**。碳当量越高，焊接冷裂纹倾向越大，需要预热/后热等控制措施。

### 原理与公式

工具同时计算三种碳当量指标：

**CE(IIW) — 国际焊接学会公式（适用于中高碳钢）：**
```
CE(IIW) = C + Mn/6 + (Cr + Mo + V)/5 + (Ni + Cu)/15
```

**CEQ(JIS) — 日本工业标准公式：**
```
CEQ = C + Mn/6 + Si/24 + Ni/40 + Cr/5 + Mo/4 + V/14
```

**Pcm — 冷裂敏感系数（Ito-Bessyo公式，适用于低碳微合金钢）：**
```
Pcm = C + Si/30 + (Mn + Cu + Cr)/20 + Ni/60 + Mo/15 + V/10 + 5B
```
注：B（硼）输入单位为 ppm，代码内部已转换为百分比（÷10000）。

**焊接性评价标准（基于 CE(IIW)）：**

| CE 范围 | 评价 |
|---------|------|
| ≤ 0.40 | 焊接性能优秀，一般不需预热 |
| 0.40 ~ 0.60 | 焊接性能良好，需采取控制措施 |
| 0.60 ~ 0.80 | 焊接困难，需严格焊接工艺 |
| > 0.80 | 极难焊接，通常不建议焊接 |

### 应用场景
- 焊接工艺评定（WPS）中的预热温度确定
- 钢材选材时的焊接性评估
- 管线钢、桥梁钢、压力容器钢的质量控制

### ⚠️ 代码问题
**HTML页面显示的公式与代码实际计算不匹配**：
1. 页面显示的"CEV"公式实际写的是 Pcm 公式形式，且出现 `DOUDLE` 拼写错误（应为 DOUBLE 或根本不需要此前缀）
2. 页面显示的"PCM"公式写成了 `C + Mn/6 + Ni/15 + Cr/5 + Cu/15 + Mo/5 + V/5`，这与代码实际计算的 Pcm 完全不同
3. **代码中三个公式的实际计算是正确的**，仅页面展示文字有误，会误导使用者

## 3. 淬透性与Jominy曲线计算器 (hardenJominy)

### 用途
基于钢材化学成分，计算**理想临界直径 DI**、预测**末端淬火（Jominy）硬度曲线**，并评估圆棒工件的**淬硬深度分布**。这是热处理工艺设计中最核心的工具之一。

### 原理与公式

工具包含三个子模块：

#### 模块一：淬透性基础参数

**初始硬度 IH（100%马氏体硬度，ASTM A255多项式）：**
```
IH = 33.087 + 50.723C + 33.662C² − 2.7048C³ − 107.02C⁴ + 43.523C⁵  (HRC)
```

**理想临界直径 DI（ASTM A255）：**
```
DI = 25.4 × f(C) × √(f_Mn × f_Si × f_Cr × f_Ni × f_Mo) × f_B  (mm)
```
各元素乘法因子：
- f_Mn = 1 + 3.34Mn
- f_Si = 1 + 0.7Si
- f_Cr = 1 + 2.16Cr
- f_Ni = 1 + 0.363Ni
- f_Mo = 1 + 3.0Mo
- f(C) 为碳含量分段函数（5段）

碳因子 f(C) 分段定义：
| C 范围 | f(C) |
|--------|------|
| C ≤ 0.39 | 0.54C |
| 0.39 < C ≤ 0.55 | 0.171 + 0.001C + 0.265C² |
| 0.55 < C ≤ 0.65 | 0.115 + 0.268C − 0.038C² |
| 0.65 < C ≤ 0.75 | 0.143 + 0.2C |
| C > 0.75 | 0.062 + 0.409C − 0.135C² |

硼因子 f_B（仅在 0 < B ≤ 0.005% 时有效）：
```
f_B = 1 + (1 − 1.5C) × min(1, 0.5 + 0.5/√alloyFactor) × (B/0.002)
f_B 上限为 2.5
```

**马氏体转变温度（Andrews公式）：**
```
Ms = 539 − 423C − 30.4Mn − 17.7Ni − 12.1Cr − 7.5Mo  (°C)
Mf = Ms − 215  (°C)
```

#### 模块二：Jominy末端淬火曲线

使用指数衰减模型：
```
HRC(J) = HRC_base + (IH − HRC_base) × e^(−k·J)
```
- J：距淬火端的距离（mm）
- k = 0.06 / √(DI/10)，衰减系数
- HRC_base = 0.2 × IH（完全退火态硬度）
- DI 越大，k 越小，硬度衰减越慢（淬透性越好）

#### 模块三：淬硬深度预测（Lamont-Grossman方法）

```
Dc = DI × (1 − e^(−1.5H))  (临界直径)
```
- H：淬火介质烈度（水=1.0, 盐水=2.0, 快速油=0.7, 中速油=0.5, 空气=0.02）

圆棒各径向位置的硬度通过等效Jominy距离查曲线获得。

### 应用场景
- 钢种选择：根据零件尺寸反推所需DI，选择合适钢种
- 热处理工艺设计：预测淬火后硬度分布
- 质量控制：对比实测Jominy曲线与预测曲线

### 公式验证
代码中的公式与 ASTM A255 / Andrews 标准公式一致，**无重大公式错误**。

## 4. 硬度转化工具 (hardness)

### 用途
在四种常用硬度标尺（努氏HK、维氏HV、布氏HB、洛氏HRC）之间进行互相转换。

### 原理与公式

基于 **ISO 18265** 标准的经验回归公式，部分典型转换如下：

| 转换方向 | 公式 | 适用范围 |
|----------|------|----------|
| HV → HRC | 116 − 1500/√HV | HRC 20~65 |
| HB → HRC | 116 − 1460/√HB | HRC 20~65 |
| HRC → HV | 0.16x² + 13.5x + 113 | HRC 30~60 |
| HRC → HB | 0.15x² + 12.8x + 107 | HRC 30~60 |
| HK → HV | HK × 0.962 | 全范围 |
| HV → HB | HV × 0.95 | HV < 600 |
| HK → HRC | 115 − 215000/HK^1.2 | HK > 250 |

### 应用场景
- 不同检测设备间的硬度值互换
- 质检报告中硬度标尺的统一
- 材料力学性能的快速估算（如 HB ≈ 3.45 × σ_b）

### 公式验证
公式为经验回归模型，在各自标注的适用范围内与ISO 18265查表数据吻合良好。**无重大公式问题。**

## 5. 密度计算 (midu)

### 用途
根据合金元素含量和温度，计算钢铁材料的**密度**。

### 原理与公式

**室温合金密度修正：**
```
ρ₂₀ = 7.85 + Σ(ki × Ci)  (g/cm³)
```
- 7.85 为纯铁基础密度
- ki 为各元素的密度影响系数，Ci 为元素含量（wt%）

代码中的 ki 值：

| 元素 | ki (g/cm³ per 1%) | 效果 |
|------|-------------------|------|
| Cr | +0.03 | 增大 |
| Ni | +0.012 | 增大 |
| Mo | +0.015 | 增大 |
| Mn | +0.005 | 增大 |
| W  | +0.1 | 显著增大 |
| Al | −0.05 | 减小 |
| Si | −0.008 | 减小 |
| C  | −0.06 | 减小 |

**高温密度修正（热膨胀）：**
```
ρ(T) = ρ₂₀ / [1 + 3α(T − 20)]
```
- α = 1.2×10⁻⁵ /°C（线膨胀系数）
- 3α 是体膨胀近似

### 应用场景
- 钢材理论重量计算（配合"钢材重量计算"工具使用）
- 铸造/锻造工艺的充型计算
- 高温下的热力学/传热计算

### ⚠️ 代码问题（需确认）
**Cr 的 ki 系数符号存疑**：纯 Cr 密度 7.19 g/cm³ 低于 Fe 的 7.87 g/cm³，理论上添加 Cr 应降低合金密度（ki 为负值）。多数参考文献给出 Cr 的 ki ≈ −0.03。代码中使用 **+0.03** 可能使高铬钢（如不锈钢）的密度计算结果偏高。建议对照厂内使用的参考手册确认此系数。

## 6. 淬火温度计算器 (quenchTemCal)

### 用途
根据碳含量和钢种类别，计算**临界温度（Ac1、Ac3）**和推荐淬火温度范围；高级模式可计算**马氏体转变温度（Ms/Mf点）**。

### 原理与公式

**基础计算模块：**
```
Ac1 = 727°C（固定值，钢铁相变基本常数）
Ac3 = 912 − 203√C  (°C)（亚共析钢经验公式）
```

淬火温度推荐逻辑：
- 亚共析钢（C < 0.77%）：Ac3 + 30~50°C
- 共析钢（C = 0.77%）：Ac1 + 30~50°C
- 过共析钢（C > 0.77%）：Ac1 + 30~50°C

合金钢温度上调：低合金+20°C，中合金+30°C，高合金+40°C

**高级模块 — Ms/Mf 计算（Andrews公式）：**
```
Ms = 539 − 423C − 30.4Mn − 17.7Ni − 12.1Cr − 7.5Mo  (°C)
Mf = Ms − 200  (°C)
```

### 应用场景
- 新钢种热处理工艺开发
- 淬火温度参数的快速查询
- Ms 点评估：判断是否需要分级淬火或等温淬火

### ⚠️ 代码问题
代码计算了推荐淬火温度范围 `tempRange`（如"787~807°C"），但**未在输出区域显示**——用户只能看到 Ac3 温度，看不到推荐的淬火温度范围，降低了工具的实用价值。

## 7. 单位制度转化 (unit)

### 用途
冶金行业常用的**七大物理量**（温度、长度、面积、体积、力、压力、质量）的单位互转。

### 原理与公式

覆盖 7 类共 28 个单位之间的双向转换，总计约 80 个转换规则。核心转换举例：

| 类别 | 转换示例 | 公式 |
|------|----------|------|
| 温度 | °C → °F | F = C × 9/5 + 32 |
| 温度 | °C → K | K = C + 273.15 |
| 长度 | mm → in | in = mm / 25.4 |
| 压力 | MPa → psi | psi = MPa × 145.038 |
| 压力 | MPa → ksi | ksi = MPa × 0.145038 |
| 力 | kN → tf | tf = kN / 9.81 |
| 质量 | kg → lb | lb = kg × 2.20462 |

### 应用场景
- 国际贸易中公制/英制单位转换
- 对照国外标准时的单位换算
- 日常冶金工程计算辅助

### ⚠️ 代码BUG（严重）
**面积转换 mm² → m² 公式错误**：
```javascript
'mm2→m2': value => value / 10000   // ← 错误！
'm2→mm2': value => value * 10000   // ← 错误！
```
正确关系：**1 m² = 1,000,000 mm²**（10⁶），当前代码除以 10,000（10⁴），**结果会偏大 100 倍**。应修改为：
```javascript
'mm2→m2': value => value / 1000000
'm2→mm2': value => value * 1000000
```

## 8. 钢材重量计算 (weight)

### 用途
根据钢材的**截面形状参数、长度和密度**，计算各类型材/板材的理论重量。

### 原理与公式

支持 15 种截面类型，核心公式为 **W = V × ρ / 1000**（体积×密度，mm³→g）

各截面体积公式：

| 钢材类型 | 体积公式 V (mm³) |
|----------|------------------|
| 圆钢 | π(d/2)² × L |
| 方钢 | d² × L |
| 钢板/扁钢 | b × d × L |
| 六角钢 | 0.866 × s² × L |
| 八角钢 | 0.828 × s² × L （**代码实际写为 8.28**） |
| 螺纹钢 | πd²/4 × L |
| 等边角钢 | [d(b−d)×2 + d² + (1−π/4)R²] × L |
| 槽钢 | [hb − (h−2t)(b−d) + 2(1−π/4)(R²−r²)] × L |
| 工字钢 | [hb − (h−2t)(b−2d) + (R²−r²)(4−π)] × L |
| H型钢 | [hb − (h−2t)(b−2d) + R²(4−π)] × L |
| 钢管 | π/4 × (D² − (D−2s)²) × L |
| 不等边角钢 | [d(b+B−d) + R²(1−π/4)] × L |
| 方管 | [w₁w₂ − (w₁−2c)(w₂−2c)] × L |
| 花纹板 | b × d × L × 1.0703 |

### 应用场景
- 钢材采购的理论重量核算
- 结构设计中的自重计算
- 材料成本估算

### ⚠️ 代码BUG（严重）
**八角钢体积系数错误**：
```javascript
volumeFormula: (s, L) => Math.pow(s, 2) * L * 8.28  // ← 错误！
```
正八边形（s 为对边距）截面积 = 2(1+√2) × [s·tan(π/8)]² ≈ **0.8284 × s²**。

当前系数 **8.28** 是正确值 **0.828** 的 10 倍（小数点位置错误），**计算出的八角钢重量会偏大 10 倍**。

## 9. 供氧强度/吹炼节奏计算器

### 用途
计算转炉吹炼的**平均氧气流量 Q** 和**供氧强度 I**，用于评估"吹得猛不猛"。支持按总耗氧量或分段吹炼计划两种模式。

### 原理与公式

```
Q = V / t          (Nm³/min，平均氧气流量)
I = Q / T          (Nm³/min/t，吨钢供氧强度)
```
- V：总耗氧量（Nm³）
- t：吹炼时间（min）
- T：炉次吨位（t）

分段模式：
```
V = Σ(Qi × ti)    (各段流量×各段时间的累加)
t = Σti
```

### 应用场景
- 转炉吹炼工艺参数设计与对比
- 不同吨位/炉型间的供氧强度横向比较
- 吹炼节奏（分段流量）的方案优化

### 公式验证
纯量纲计算，**无公式问题**。代码在 I < 3 或 I > 12 时给出经验区间告警，合理。

## 10. 冷却剂/矿石温降计算器

### 用途
基于能量守恒，估算"**加多少冷却剂能降多少温**"或"**要降到目标温度需要加多少冷却剂**"。

### 原理与公式

**钢水需要释放的热量：**
```
Q_need = m_steel(kg) × Cp_steel(kJ/kg·K) × (T_current − T_target)
```

**冷却剂每 kg 吸收的热量 q：**

仅加热模式（矿石/氧化物）：
```
q = Cp_s × (T_final − T₀) + H_reaction
```

加热+熔化+过热模式（废钢/金属料）：
```
q = Cp_s × (Tm − T₀) + L + Cp_l × (T_final − Tm) + H_reaction
```
- Cp_s：固相比热，Cp_l：液相比热
- Tm：熔点，L：熔化潜热
- H_reaction：反应吸/放热（正=吸热，负=放热）

## 13. 连铸壳厚/拉速安全核算

### 用途

```
t = L / v  (min)
```

```
v_max = L × (K / δ_min)²  (m/min)
```

```
L_req = v × (δ_min / K)²  (m)
```

## 15. 轧制力计算器 (rollingForce)

### 用途
计算热轧过程中的**轧制力 P**，这是轧钢工艺设计最核心的参数——它决定了轧机能力、轧辊强度、道次分配方案。

### 原理与公式

**压下量与压下率：**
```
Δh = H − h  (mm)
ε = Δh / H × 100%
```
- H：入口厚度，h：出口厚度

**接触弧长（几何关系）：**
```
l_d = √(R × Δh)  (mm)
```
- R：工作辊半径（mm）

**应力状态系数（Ekelund 公式，热轧经典公式）：**
```
Q_p = 1 + 1.6μ√(RΔh)/(H+h) − 1.2Δh/(H+h)
```
- μ：轧辊-轧件之间的摩擦系数
- 第二项为摩擦引起的应力增大效应
- 第三项为压下引起的应力缓解效应

**轧制力：**
```
P = K_f × Q_p × b × l_d / 1000  (kN)
```
- K_f：变形抗力（MPa）
- b：轧件宽度（mm）
- 除以 1000 将 N 转为 kN

### 默认值（热轧中板典型工况）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| R | 450 mm | 热轧宽带钢工作辊半径 |
| H | 30 mm | 中间坯厚度 |
| h | 22 mm | 精轧某道次出口厚度 |
| b | 1200 mm | 板宽 |
| K_f | 150 MPa | 低碳钢 1000°C 左右典型值 |
| μ | 0.35 | 热轧典型摩擦系数 |

### 应用场景
- 热轧板带轧制力估算——确定轧机能力和道次分配
- 型钢轧制——估算各道次轧制力，校核轧辊强度
- 棒线材轧制——粗轧、中轧、精轧各道次计算
- 设备选型——根据最大轧制力选择轧机规格

### 公式验证
Ekelund 公式为热轧工程计算的经典公式，代码实现与文献一致。**无公式问题。**

## 16. 压下规程计算器 (reductionSchedule)

### 用途
设计多道次轧制的**压下分配方案**。给定坯料厚度和成品厚度，自动分配各道次的压下量，是轧制工艺设计的基础工作。

### 原理与公式

**等压下率分配模式：**
```
ε_avg = 1 − (h_f / H₀)^(1/n)
```
- H₀：坯料厚度（mm），h_f：成品厚度（mm），n：道次数
- 每道次使用相同的压下率 ε_avg
- 各道次出口厚度：h_i = h_{i-1} × (1 − ε_avg)

**递减压下率分配模式：**
```
权重 w_i = ln(n − i + 2)    （i = 1, 2, ..., n）
ε_i = ε_avg × (w_i / w_avg)
```
- 前面道次压下率大，后面道次逐渐减小
- 使用对数权重实现平滑递减

**各道次参数：**
```
Δh_i = H_i − h_i          (压下量)
ε_i = Δh_i / H_i × 100%   (压下率)
λ_i = H_i / h_i            (延伸系数)
```

**汇总参数：**
```
总延伸系数 λ_total = H₀ / h_f
总压下率 = (1 − h_f/H₀) × 100%
```

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| H₀ | 230 mm | 连铸坯/板坯厚度 |
| h_f | 12 mm | 成品板厚 |
| n | 7 | 精轧道次数 |
| 分配方式 | 等压下率 | 工程中最常用的起始方案 |

### 应用场景
- 热轧板带精轧机组的道次分配设计
- 粗轧/中轧的压下方案初步估算
- 不同分配策略（等压下率 vs 递减）的效果对比

### 公式验证
代码中的等压下率和递减压下率分配算法数学推导正确。**无公式问题。**

## 18. 变形抗力计算器 (deformResistance)

### 用途
根据钢种（碳含量）和轧制工况（温度、应变、应变速率），计算热轧过程的**变形抗力 K_f**。K_f 是轧制力计算的关键输入参数。

工具同时提供两种经典模型的计算结果，供对比使用。

**Misaka 公式（碳钢热轧最常用）：**

```
K_f = exp(0.126 − 1.75C + 0.594C² + (2851 + 2968C − 1120C²)/T_K) × ε^n × ε̇^m × 9.81  (MPa)
```

- T_K = T(°C) + 273.15（绝对温度）

- 9.81 为单位转换因子（kgf/mm² → MPa）

**Shida 公式：**
```
K_f = 0.28 × exp(5000/T_K − 0.01T_K) × f(C) × ε^n × ε̇^m × 9.81  (MPa)
```
- f(C) = 1 + (1.3C − 0.3C²)（碳含量修正）
- n = 0.41 − 0.07C
- m = −0.019 + 0.126/(C + 0.05) × T_K/10000

| 参数 | 默认值 | 说明 |
|------|--------|------|
| C | 0.15% | 低碳钢（Q235/SS400 级） |

### 应用场景
- 为轧制力计算器提供 K_f 输入值

- 两种模型的对比验证

```
α ≤ β    →  ✅ 可以咬入
α > β    →  ❌ 无法咬入
```

**最大允许压下量：**
```
Δh_max = 2R × (1 − cos(β))  (mm)
```

**热轧摩擦系数经验公式（按温度自动计算模式）：**
```
μ = 0.8 × (1.05 − 0.0005T)
```

## 20. 轧制温降计算器 (rollingTempDrop)

### 用途
估算热轧过程中的**温度变化**——辐射散热导致的温降和变形产热导致的温升，最终给出轧件经过一个道次后的温度。温度控制是热轧工艺的核心环节。

### 原理与公式

**辐射温降（Stefan-Boltzmann 定律简化）：**
```
ΔT_rad = σ × ε_r × (T₀_K⁴ − T_env_K⁴) × t / (ρ × c × h_m / 2)  (°C)
```
- σ = 5.67×10⁻⁸ W/(m²·K⁴)（Stefan-Boltzmann 常数）
- ε_r = 0.8（钢的辐射率）
- T₀_K：初始温度的绝对温度（K）
- T_env_K = 293.15 K（环境温度 20°C）
- t：空冷时间（s）
- ρ = 7850 kg/m³（钢密度）
- c = 680 J/(kg·°C)（钢比热容）
- h_m = h/1000（轧件厚度，mm→m）

**变形升温：**
```
ΔT_def = η × K_f × 10⁶ × ε / (ρ × c)  (°C)
```
- η = 0.90（变形功转化为热的效率）
- K_f：变形抗力（MPa），×10⁶ 转为 Pa
- ε：应变

**最终温度：**
```
T_final = T₀ − ΔT_rad + ΔT_def  (°C)
```

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| T₀ | 1050°C | 粗轧出口/精轧入口温度 |
| h | 25 mm | 轧件厚度 |
| b | 1200 mm | 轧件宽度（参考） |
| t | 8 s | 道次间空冷时间 |
| K_f | 150 MPa | 变形抗力 |
| ε | 0.3 | 单道次应变 |

### 应用场景
- 精轧入口/出口温度预测
- 道次间冷却策略优化
- 终轧温度控制方案设计

### 公式验证
基于 Stefan-Boltzmann 辐射定律和能量守恒的简化模型，公式推导正确。注意：该模型为工程估算，未考虑对流散热、接触散热（辊冷）、水冷等因素。**无公式问题。**

## 21. 前滑值与出口速度计算器 (forwardSlip)

### 用途
计算轧件出辊缝时相对于辊面线速度的**前滑值 S_h** 和实际的**出口速度 v_out**。前滑是连轧速度匹配和测厚仪标定的基础参数。

### 原理与公式

**压下量：**
```
Δh = H − h  (mm)
```

**前滑值（Ekelund 近似）：**
```
S_h = Δh / (4h) × √(h/R)
```
- H：入口厚度，h：出口厚度，R：工作辊半径

**出口速度：**
```
v_out = v_R × (1 + S_h)  (m/s)
```
- v_R：辊面线速度（m/s）

**物理意义：**
- 前滑与压下率正相关——压下量越大，前滑越大
- 前滑与辊径负相关——辊径越大，前滑越小
- 出口速度始终大于辊面线速度

| 参数 | 默认值 | 说明 |

| R | 450 mm | 工作辊半径 |
| H | 30 mm | 入口厚度 |
| h | 22 mm | 出口厚度 |
| v_R | 3.5 m/s | 辊面线速度 |

### 应用场景
- 连轧速度控制——各机架速度匹配的基础
- 测厚仪标定——利用前滑关系校准在线测厚
- 活套控制——机架间张力与活套高度调节

## 22. 热膨胀/冷缩量计算器 (thermalExpansion)

### 用途
计算钢材从热态冷却到室温后的**尺寸收缩量**，用于从热态测量尺寸推算冷态成品尺寸。这是热轧成品尺寸精度控制的基础。

### 原理与公式

**冷态尺寸：**
```
L_cold = L_hot / (1 + α × (T_hot − T_cold))  (mm)
```
- L_hot：热态测量尺寸（mm）
- T_hot：热态温度（°C）
- T_cold：冷态温度（°C），通常取室温 20°C
- α = 12.5 × 10⁻⁶ /°C（钢的线膨胀系数）

**收缩量与收缩率：**
```
ΔL = L_hot − L_cold  (mm)
收缩率 = ΔL / L_hot × 100%
```

**工程经验：**
- 900°C 时收缩率约 1.1%
- 即热态 1000mm 的板材冷到室温约缩短 11mm

### 默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| L_hot | 1000 mm | 热态尺寸 |
| T_hot | 900°C | 终轧/卷取温度 |
| T_cold | 20°C | 室温 |

### 应用场景
- 热轧成品宽度/厚度的冷态换算
- 精轧辊缝设定时预留收缩余量
- 在线测厚/测宽仪的热态→冷态修正

### 公式验证
线膨胀公式为经典热力学公式，α = 12.5×10⁻⁶ /°C 是碳钢在 20~900°C 范围内的平均值，工程上广泛采用。**无公式问题。**
